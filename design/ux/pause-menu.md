# Pause Menu UX Design

> **Status**: Approved
> **Author**: fanioz + ux-designer
> **Last Updated**: 2026-06-11
> **Template**: UX Spec

---

## 1. Goal
Provide a clean, interruption-free menu for players to adjust settings, restart the level, or exit the game. Must maintain the "Neon Cipher" aesthetic.

## 2. Layout
Centered overlay box over a blurred/darkened puzzle board.
- Resume Game
- Restart Level
- Settings (Volume, Colorblind Mode)
- Exit to Title

## 3. Dynamic Behaviors
- **Entrance**: Fades in over 0.2s, applying a heavy blur to the game world.
- **Exit**: Fades out instantly on Resume.
