# Project Context

Solstice Cipher is a puzzle game built specifically for the June Solstice Game Jam hackathon. It features unique cipher decryption mechanics, tools like splitters, and immersive logic puzzles.

# Key Features

- Built for the June Solstice Game Jam
- Laser and optical tool puzzle mechanics (splitters, rotation, briefcase inventory)
- Satisfying visual feedback and decryption logic
