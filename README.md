## About the Game

**Solstice Cipher** is a sci-fi optical puzzle game where you manipulate light beams to decrypt hidden codes. You are given a grid and a briefcase full of optical tools like mirrors and beam splitters. Your goal is to route a single light emitter through the correct sequence of receiver glyphs to reveal the hidden cipher.

It features a sleek, high-contrast neon aesthetic with strict orthogonal grids, capturing the feeling of classic "laser reflecting" games with modern polish and smooth animations.

## Features

* **Optical Puzzles:** Drag and drop tools like mirrors and splitters to direct light beams around obstacles.
* **Contextual Learning:** Learn the mechanics naturally as you play, thanks to a contextual UI tutorial system that guides you without breaking immersion.
* **Neon Aesthetic:** A sleek, high-contrast visual style governed by the rule: *"Luminescence is function; everything else is the void."*
* **Cross-Platform:** Available to play instantly on the Web and exported as an Android App Bundle (AAB).

## How to Play

1. **Equip Tools:** Drag optical tools (like mirrors or splitters) from your briefcase onto the grid.
2. **Rotate:** Click on the tools on the board to rotate them and change the beam's direction.
3. **Decrypt:** Route the light beam to hit the correct receiver glyphs in order to decrypt the code and complete the level.

## Development

This game was built using **Godot 4** (Compatibility Renderer) specifically for the **June Solstice Game Jam**. 

### Local Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/fanioz/solstice-cipher.git
   ```
2. Open the project folder (`solstice-cipher`) using **Godot 4.6.3** or newer.
3. Run the game by pressing `F5` (Play Project).
