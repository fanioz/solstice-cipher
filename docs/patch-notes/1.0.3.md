# Patch 1.0.3 — Welcome to the Cipher
*2026-06-16*

## Highlights
We're so excited to see everyone jumping into Solstice Cipher! To make the learning curve a little smoother, we've added some brief, on-screen tutorials to get you comfortable with the optical tools.

## Quality of Life
- **Interactive Tutorials**: We've added clear, non-intrusive tutorial text to Level 1 and Level 6.
  - In Level 1, you'll now see instructions on how to drag your first Mirror onto the board.
  - Once you drop the Mirror, a new tooltip will pop up explaining how to grab the outer ring to rotate it!
  - In Level 6, similar instructions will guide you on using the Prism to split the beam.
